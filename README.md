# Curso de Linguagem C

Repositório com minhas anoações e exercícios realizados durante o curso do Prof. Dr. André Backes, da FACOM/UFU.

Link do curso: https://www.facom.ufu.br/~backes/index.php  

---

## Sobre o curso

O curso possui **123 videoaulas** sobre Linguagem C e Estruturas de Dados.

Como já tenho conhecimento prévio de parte do conteúdo, não assistirei a todas as aulas.  
A ideia é focar apenas nos tópicos que ainda não domino ou que preciso reforçar.

Além das aulas, o curso conta com:
- 11 listas de exercícios de **Linguagem C**
- 9 listas de exercícios de **Estruturas de Dados em C**

Pretendo resolver **pelo menos metade dos exercícios de cada lista**, podendo pular aqueles cujo conteúdo eu já domine.
